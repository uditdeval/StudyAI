# 🚀 StudyAI — Intelligent Study Assistant

🌐 **Live Demo:** https://uditdeval.github.io/StudyAI/

---

## 📌 About

StudyAI is a lightweight AI-inspired study assistant built using HTML, CSS, and JavaScript.

It helps students quickly understand topics with clear explanations and concise key points — no need to scroll through long articles. Perfect for quick revision and concept clarity.

---

## ✨ Features

* 🔍 Smart topic search with keyword matching
* 🎯 Instant explanations with key takeaways
* 🎨 Clean and modern dark-themed UI
* 📚 Covers CS, Math, Physics & Biology basics
* ⚡ Fast and lightweight (no backend required)
* 📱 Fully responsive design

---

## 🛠 Tech Stack

* HTML5
* CSS3 (Custom + Bootstrap 5)
* JavaScript (Vanilla)

---

## 🚀 Use Case

Ideal for students who want:

* Quick concept revision before exams
* Easy-to-understand explanations
* A distraction-free study experience
